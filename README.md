### O que foi utilizado

- Vite version 4.1.0;

- Typescript version 4.9.3;

- React Router DOM;

- React Context;

- Material UI;

- Node version 18.5.0;



### Rodar projeto


#### Clone repositório

`git clone https://github.com/alessandradocouto/plant-store-landing-page.git`



#### Entar na pasta do projeto e instalar pacotes

`cd plant-store-landing-page`

`npm install`



#### Start the app

`npm run dev`
